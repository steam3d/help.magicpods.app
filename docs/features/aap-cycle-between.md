### Cycle between

Press the button on AirPods Max or press and hold the left or right AirPod on AirPods Pro to cycle between the selected noise control modes

| Option             | Description                    |
| ------------------ | ------------------------------ |
| Off                | —                              |
| Transparency       | —                              |
| Adaptive           | AirPods Pro 2, AirPods 4 (ANC) |
| Noise cancellation | -                              |
