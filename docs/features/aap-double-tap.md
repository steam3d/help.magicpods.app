### Double tap left AirPod

Action when you double-tap an AirPod headphone.

| Option         | Description |
| -------------- | ----------- |
| Siri           | —           |
| Play/Pause     | —           |
| Next track     | —           |
| Previous track | -           |
| Off            | -           | 

Siri does not work on Windows.
